# data/

## Responsibility
This directory contains BibTeX bibliography files that manage technical skills and publications data for the resume/CV. These files provide structured data that can be processed by LaTeX document generation systems to render the final CV.

## Files

- **skills.bib**: Technical skills bibliography. Contains skill entries with proficiency levels (Python, Go, Cloud Computing, etc.) and a reference about MongoDB published in Hacker magazine (2013).

- **publications.bib**: Academic publications bibliography. Contains commented academic article references and a conference proceeding entry about MongoDB by the author.

- **codemap.md**: This documentation file describing the directory structure and contents.
